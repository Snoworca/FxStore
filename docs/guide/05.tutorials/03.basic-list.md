## 학습 목표

- List 생성/열기
- 인덱스 기반 접근
- 요소 추가/삭제
- 검색 및 순회

## 1. List 생성 및 기본 연산

```java
import com.snoworca.fxstore.api.FxStore;
import java.nio.file.Paths;
import java.util.List;

public class ListBasics {
    public static void main(String[] args) {
        FxStore store = FxStore.open(Paths.get("logs.fx"));

        // List 생성
        List<String> logs = store.createOrOpenList("logs", String.class);

        // 요소 추가 (끝에)
        logs.add("Application started");
        logs.add("User logged in");
        logs.add("Data processed");

        // 크기 확인
        System.out.println("Size: " + logs.size());  // 3

        // 인덱스로 접근
        String first = logs.get(0);  // "Application started"
        String second = logs.get(1);  // "User logged in"

        // 전체 출력
        for (int i = 0; i < logs.size(); i++) {
            System.out.println(i + ": " + logs.get(i));
        }

        store.close();
    }
}
```

## 2. 요소 추가

```java
List<String> items = store.createOrOpenList("items", String.class);

// 끝에 추가
items.add("Item 1");
items.add("Item 2");

// 특정 위치에 삽입
items.add(1, "Inserted");  // [Item 1, Inserted, Item 2]

// 여러 요소 추가
items.addAll(Arrays.asList("Item 3", "Item 4"));
```

## 3. 요소 수정 및 삭제

```java
// 수정
items.set(0, "Modified Item 1");

// 인덱스로 삭제
String removed = items.remove(1);  // "Inserted" 반환

// 값으로 삭제 (첫 번째 일치)
items.remove("Item 3");

// 전체 삭제
items.clear();
```

## 4. 검색

```java
List<String> names = store.createOrOpenList("names", String.class);
names.add("Alice");
names.add("Bob");
names.add("Alice");  // 중복 허용
names.add("Charlie");

// 포함 여부
boolean hasAlice = names.contains("Alice");  // true

// 첫 번째 인덱스
int first = names.indexOf("Alice");  // 0

// 마지막 인덱스
int last = names.lastIndexOf("Alice");  // 2

// 없는 경우
int notFound = names.indexOf("Unknown");  // -1
```

## 5. 부분 리스트

```java
// subList (뷰)
List<String> sub = names.subList(1, 3);  // [Bob, Alice]
```

## 전체 예제

```java
import com.snoworca.fxstore.api.FxStore;
import java.nio.file.Paths;
import java.util.List;

public class ListTutorial {
    public static void main(String[] args) {
        FxStore store = FxStore.open(Paths.get("list_tutorial.fx"));

        // 활동 로그
        List<String> activities = store.createOrOpenList(
            "activities", String.class);

        // 로그 추가
        activities.add("[10:00] Started");
        activities.add("[10:05] Processing");
        activities.add("[10:10] Completed");

        // 최근 로그 조회
        int size = activities.size();
        if (size > 0) {
            System.out.println("Latest: " + activities.get(size - 1));
        }

        // 전체 로그 출력
        System.out.println("All activities:");
        for (String activity : activities) {
            System.out.println("  " + activity);
        }

        store.close();
    }
}
```
