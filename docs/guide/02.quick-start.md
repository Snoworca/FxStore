# 빠른 시작

> **예상 시간**: 5분
> **난이도**: 초급

이 가이드에서는 FxStore를 사용하여 데이터를 저장하고 조회하는 첫 번째 프로그램을 작성합니다.

## 1. 의존성 추가

### Gradle

```groovy
dependencies {
    implementation 'com.snoworca:fxstore:0.3.0'
}
```

### Maven

```xml
<dependency>
    <groupId>com.snoworca</groupId>
    <artifactId>fxstore</artifactId>
    <version>0.3.0</version>
</dependency>
```

## 2. 첫 프로그램

```java
import com.snoworca.fxstore.api.FxStore;
import java.nio.file.Paths;
import java.util.NavigableMap;

public class QuickStart {
    public static void main(String[] args) {
        // 1. Store 열기
        FxStore store = FxStore.open(Paths.get("mydata.fx"));

        // 2. Map 생성
        NavigableMap<Long, String> users = store.createOrOpenMap(
            "users", Long.class, String.class);

        // 3. 데이터 저장
        users.put(1L, "Alice");
        users.put(2L, "Bob");
        users.put(3L, "Charlie");

        // 4. 데이터 조회
        System.out.println("User 1: " + users.get(1L));  // Alice
        System.out.println("User 2: " + users.get(2L));  // Bob

        // 5. 전체 데이터 출력
        System.out.println("All users:");
        for (var entry : users.entrySet()) {
            System.out.println("  " + entry.getKey() + ": " + entry.getValue());
        }

        // 6. Store 닫기
        store.close();

        System.out.println("\nData saved to mydata.fx");
    }
}
```

실행 결과:
```
User 1: Alice
User 2: Bob
All users:
  1: Alice
  2: Bob
  3: Charlie

Data saved to mydata.fx
```

## 3. 재시작 후 데이터 확인

프로그램을 다시 실행하면 이전에 저장한 데이터가 그대로 남아있습니다.

```java
import com.snoworca.fxstore.api.FxStore;
import java.nio.file.Paths;
import java.util.NavigableMap;

public class VerifyData {
    public static void main(String[] args) {
        // Store 열기 (기존 파일)
        FxStore store = FxStore.open(Paths.get("mydata.fx"));

        // 기존 Map 열기
        NavigableMap<Long, String> users = store.openMap(
            "users", Long.class, String.class);

        // 데이터 확인
        System.out.println("Found " + users.size() + " users");
        System.out.println("User 1: " + users.get(1L));  // Alice

        store.close();
    }
}
```

## 4. 메모리 모드 (테스트용)

파일 없이 메모리에서만 동작하는 모드도 있습니다.

```java
import com.snoworca.fxstore.api.FxStore;
import java.util.NavigableMap;

public class MemoryMode {
    public static void main(String[] args) {
        // 메모리 모드로 Store 열기
        FxStore store = FxStore.openMemory();

        NavigableMap<String, String> config = store.createOrOpenMap(
            "config", String.class, String.class);

        config.put("theme", "dark");
        config.put("language", "ko");

        System.out.println("Theme: " + config.get("theme"));

        store.close();
        // Store가 닫히면 데이터는 사라짐
    }
}
```

## 5. 다른 컬렉션 사용

FxStore는 Map 외에도 Set, List, Deque를 지원합니다.

```java
import com.snoworca.fxstore.api.FxStore;
import java.nio.file.Paths;
import java.util.*;

public class OtherCollections {
    public static void main(String[] args) {
        FxStore store = FxStore.open(Paths.get("collections.fx"));

        // Set: 고유 요소 저장
        NavigableSet<String> tags = store.createOrOpenSet("tags", String.class);
        tags.add("java");
        tags.add("database");

        // List: 순서가 있는 목록
        List<String> logs = store.createOrOpenList("logs", String.class);
        logs.add("Started");
        logs.add("Processing");

        // Deque: 양방향 큐
        Deque<String> tasks = store.createOrOpenDeque("tasks", String.class);
        tasks.addLast("Task 1");
        tasks.addLast("Task 2");
        String next = tasks.pollFirst();  // "Task 1"

        store.close();
    }
}
```
