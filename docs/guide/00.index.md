## 지원 컬렉션

| 컬렉션 | 인터페이스 | 용도 |
|--------|-----------|------|
| Map | `NavigableMap<K, V>` | 키-값 저장, 정렬, 범위 조회 |
| Set | `NavigableSet<E>` | 고유 요소, 정렬, 범위 조회 |
| List | `List<E>` | 인덱스 접근, 순서 유지 |
| Deque | `Deque<E>` | 양방향 큐, Stack/Queue |

## 버전

- **현재 버전**: 0.3.0
- **Java 요구사항**: 8+
- **의존성**: 없음 (zero-dependency)
