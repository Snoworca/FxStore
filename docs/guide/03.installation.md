# 설치 가이드

> **예상 시간**: 5분
> **난이도**: 초급

## 요구사항

- **Java**: 8 이상
- **빌드 도구**: Gradle 또는 Maven (선택)

## Gradle 설정

### Kotlin DSL (build.gradle.kts)

```kotlin
dependencies {
    implementation("com.snoworca:fxstore:0.3.0")
}
```

### Groovy DSL (build.gradle)

```groovy
dependencies {
    implementation 'com.snoworca:fxstore:0.3.0'
}
```

## Maven 설정

### pom.xml

```xml
<dependency>
    <groupId>com.snoworca</groupId>
    <artifactId>fxstore</artifactId>
    <version>0.3.0</version>
</dependency>
```

## 수동 설치 (JAR 직접 사용)

1. [릴리스 페이지](https://github.com/snoworca/fxstore/releases)에서 JAR 다운로드
2. 프로젝트의 `libs/` 폴더에 복사
3. 빌드 도구에서 로컬 JAR 참조

### Gradle (로컬 JAR)

```groovy
dependencies {
    implementation files('libs/fxstore-0.3.0.jar')
}
```

### Maven (로컬 JAR)

```xml
<dependency>
    <groupId>com.snoworca</groupId>
    <artifactId>fxstore</artifactId>
    <version>0.3.0</version>
    <scope>system</scope>
    <systemPath>${project.basedir}/libs/fxstore-0.3.0.jar</systemPath>
</dependency>
```

## IDE 설정

### IntelliJ IDEA

1. **File > Project Structure > Libraries**
2. **+** 버튼 클릭 > **From Maven...**
3. `com.snoworca:fxstore:0.3.0` 검색 및 추가

또는 Gradle/Maven 동기화:
- **View > Tool Windows > Gradle** > 새로고침 버튼

### Eclipse

1. **Project > Properties > Java Build Path**
2. **Libraries** 탭 > **Add External JARs...**
3. fxstore JAR 파일 선택

Maven 프로젝트인 경우:
- **Project > Update Maven Project** (Alt+F5)

### VS Code

1. Gradle/Maven 확장 설치
2. `build.gradle` 또는 `pom.xml`에 의존성 추가
3. **Java: Clean Java Language Server Workspace** 명령 실행

## 설치 확인

설치가 완료되었는지 확인하는 간단한 코드:

```java
import com.snoworca.fxstore.api.FxStore;

public class InstallCheck {
    public static void main(String[] args) {
        // 메모리 모드로 Store 생성
        FxStore store = FxStore.openMemory();

        // 버전 확인 (0.3.0에서는 API 없음, 정상 동작 확인용)
        System.out.println("FxStore is working!");

        store.close();
    }
}
```

출력:
```
FxStore is working!
```

## 문제 해결

### ClassNotFoundException

```
java.lang.ClassNotFoundException: com.snoworca.fxstore.api.FxStore
```

**원인**: 클래스패스에 FxStore JAR이 없음

**해결**:
1. 의존성이 올바르게 추가되었는지 확인
2. Gradle/Maven 동기화 실행
3. IDE에서 프로젝트 새로고침

### NoClassDefFoundError

```
java.lang.NoClassDefFoundError: com/snoworca/fxstore/api/FxStore
```

**원인**: 런타임에 JAR을 찾을 수 없음

**해결**:
1. 런타임 클래스패스에 JAR 포함 확인
2. Gradle의 경우 `implementation` 스코프 사용
