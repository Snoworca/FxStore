# Installation Guide

> **Estimated Time**: 5 minutes
> **Level**: Beginner

## Requirements

- **Java**: 8 or higher
- **Build Tool**: Gradle or Maven (optional)

## Gradle Setup

### Kotlin DSL (build.gradle.kts)

```kotlin
dependencies {
    implementation("com.snoworca:fxstore:0.3.0")
}
```

### Groovy DSL (build.gradle)

```groovy
dependencies {
    implementation 'com.snoworca:fxstore:0.3.0'
}
```

## Maven Setup

### pom.xml

```xml
<dependency>
    <groupId>com.snoworca</groupId>
    <artifactId>fxstore</artifactId>
    <version>0.3.0</version>
</dependency>
```

## Manual Installation (Direct JAR Usage)

1. Download JAR from [releases page](https://github.com/snoworca/fxstore/releases)
2. Copy to project's `libs/` folder
3. Reference local JAR in build tool

### Gradle (Local JAR)

```groovy
dependencies {
    implementation files('libs/fxstore-0.3.0.jar')
}
```

### Maven (Local JAR)

```xml
<dependency>
    <groupId>com.snoworca</groupId>
    <artifactId>fxstore</artifactId>
    <version>0.3.0</version>
    <scope>system</scope>
    <systemPath>${project.basedir}/libs/fxstore-0.3.0.jar</systemPath>
</dependency>
```

## IDE Setup

### IntelliJ IDEA

1. **File > Project Structure > Libraries**
2. Click **+** button > **From Maven...**
3. Search and add `com.snoworca:fxstore:0.3.0`

Or sync Gradle/Maven:
- **View > Tool Windows > Gradle** > Refresh button

### Eclipse

1. **Project > Properties > Java Build Path**
2. **Libraries** tab > **Add External JARs...**
3. Select fxstore JAR file

For Maven projects:
- **Project > Update Maven Project** (Alt+F5)

### VS Code

1. Install Gradle/Maven extension
2. Add dependency to `build.gradle` or `pom.xml`
3. Run **Java: Clean Java Language Server Workspace** command

## Verify Installation

Simple code to verify installation:

```java
import com.snoworca.fxstore.api.FxStore;

public class InstallCheck {
    public static void main(String[] args) {
        // Create store in memory mode
        FxStore store = FxStore.openMemory();

        // Version check (no API in 0.3.0, just verifying it works)
        System.out.println("FxStore is working!");

        store.close();
    }
}
```

Output:
```
FxStore is working!
```

## Troubleshooting

### ClassNotFoundException

```
java.lang.ClassNotFoundException: com.snoworca.fxstore.api.FxStore
```

**Cause**: FxStore JAR not in classpath

**Solution**:
1. Verify dependency is added correctly
2. Run Gradle/Maven sync
3. Refresh project in IDE

### NoClassDefFoundError

```
java.lang.NoClassDefFoundError: com/snoworca/fxstore/api/FxStore
```

**Cause**: JAR not found at runtime

**Solution**:
1. Verify JAR is included in runtime classpath
2. Use `implementation` scope for Gradle
