## Supported Collections

| Collection | Interface | Use Case |
|------------|-----------|----------|
| Map | `NavigableMap<K, V>` | Key-value storage, sorting, range queries |
| Set | `NavigableSet<E>` | Unique elements, sorting, range queries |
| List | `List<E>` | Index access, order preservation |
| Deque | `Deque<E>` | Double-ended queue, Stack/Queue |
