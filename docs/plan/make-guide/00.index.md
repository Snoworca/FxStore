## 핵심 기능 요약

### 지원 컬렉션
- `NavigableMap<K, V>` - 정렬된 키-값 맵
- `NavigableSet<E>` - 정렬된 집합
- `List<E>` - 인덱스 기반 리스트
- `Deque<E>` - 양방향 큐

### 주요 특징
- **COW (Copy-on-Write)**: 크래시 일관성 보장
- **단일 파일**: 하나의 파일에 모든 데이터 저장
- **메모리 모드**: 테스트/임시 저장용
- **Zero-dependency**: 외부 의존성 없음
